### NodeTs - Project Dog Kennel ###
  Projeto feito no módulo 2 do curso de NodeJs + TypeScript da B7Web.

### Pré-requisitos Globais ###
  npm -i -g nodemon
  npm -i -g typescript
  npm -i -g ts-node

### Instalação ####
  Caso clone o repositório, utilizar comando => npm install.

### Execução do projeto ###
  npm run dev
